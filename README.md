# Getting Started
Requirements
git
You'll know you did it right if you can run 
```shell
git --version 
```
and you see a response like 
```shell
git version x.x.x
```
foundry
You'll know you did it right if you can run 
```shell
forge --version
```
and you see a response like 
```shell
forge 0.2.0 (816e00b 2023-03-16T00:05:26.396218Z)
```
